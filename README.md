# Goliath zero-downtime restart example

## How it works?

See https://stripe.com/blog/meet-einhorn for more background.

## Installation

```
$> bundle
```

## Running

```ruby
$> bundle exec einhorn -b 127.0.0.1:3000 ruby app.rb --einhorn
```

## If you encountered on EACESS error

```bash
> chmod +x app.rb
```
